## web_test_ambev_tech

Automação de Testes Web - Ambev Tech

---

📝 **Descrição**

Este projeto contém testes automatizados para uma aplicação web utilizando **Robot Framework**. Os testes cobrem funcionalidades básicas como login, busca de produtos e gerenciamento de lista de compras.

---

🛠️ **Tecnologias Utilizadas**

* Robot Framework
* SeleniumLibrary
* Python

---

🗂️ **Estrutura do Projeto**


├── resources/
│   ├── keywords.robot    \# Contém as keywords e variáveis do projeto
│   └── steps.robot       \# Contém os steps em Gherkin
└── test-case/
└── web-test.robot    \# Casos de teste


---

📋 **Cenários de Teste**

* Verificação de login com credenciais válidas
* Busca de produtos
* Adição de produto à lista de compras
* Acesso à lista de compras

---

⚙️ **Pré-requisitos**

* Python 3.x
* Robot Framework
* Selenium WebDriver
* Chrome Browser

---

🚀 **Como Executar**

1.  Clone o repositório
2.  Instale as dependências
3.  Execute os testes

Para executar um teste específico, use a tag correspondente:

---

🏷️ **Tags Disponíveis**

* `login`: Teste de login
* `search_product`: Teste de busca de produtos
* `Notes`: Teste de adição à lista de compras
* `open_shopping_list`: Teste de acesso à lista de compras

---

📊 **Relatórios**

Após a execução, os relatórios serão gerados automaticamente na pasta `results/`:

* `report.html`: Relatório detalhado dos testes
* `log.html`: Log detalhado da execução
* `output.xml`: Arquivo XML com os resultados
